Slide Shortcuts

  • direct on slides:
    • Pressing 'b' brings you back here
    • Pressing 'h' highlights code snippets
    • Pressing 'p' toggles speaker notes (if they're on the current slide)
    • Pressing 'f' toggles fullscreen viewing
    • Pressing 'w' toggles widescreen
    • Pressing 'o' toggles overview mode
    • Pressing 'ESC' toggles off these goodies
  • on legacy shop application:
    • Pressing 'F1' changes design to standard bootstrap
    • Pressing 'F2' toggles different bootstrap themes
    • Pressing 'F3' toggles use case highlight
    • Pressing 'F4' toggles visibility of the recommendation panel on catalog page
